## Segundo - WeMovies
O desafio consiste na criação de um site de e-commerce simplificado do zero, onde o candidato terá que criar um fluxo com três módulos/telas.

### Instruções
1. Instalar dependências
```shell
bun i
```
2. Rodar o server
```shell
bun run-server
```
3. Rodar os testes
```shell
bun run test
```
4. Rodar o app
```shell
bun dev
```
